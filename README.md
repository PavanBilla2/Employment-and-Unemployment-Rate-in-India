# 📊 India Employment & Job Market Analysis Dashboard (Power BI)

## 🧠 Project Overview

This project presents an interactive **Power BI dashboard** analyzing:

* 📉 State-wise unemployment trends in India
* 💼 Data science job market insights (salary, roles, companies)

The goal is to provide meaningful insights into employment patterns and job opportunities using real-world datasets.

---

## 📂 Datasets Used

### 1️⃣ Employment in India Dataset

* State-wise unemployment rate
* Labour participation rate
* Estimated employed population
* Time-series data

### 2️⃣ Data Science Jobs Dataset

* Job roles (Data Analyst, Data Scientist, etc.)
* Salary ranges (Min, Max, Avg)
* Company information

---

## 🎯 Key Features

### 📍 Page 1: Employment Overview

* Filled Map showing unemployment by state
* Top 10 states with highest unemployment
* Year-wise trend analysis
* KPI Cards:

  * Avg Unemployment Rate
  * Total Employed
  * Labour Participation Rate

---

### 💼 Page 2: Unemployment Overview

* Average salary by job role
* Top hiring companies
* Total job listings
* Salary insights (in Lakhs ₹)

---

### 📊 Page 3: Data Science Job Market

* Comparative analysis (employment vs job market)
* Key observations and trends
* Summary for decision-making

---

## 🛠️ Tools & Technologies

* Power BI (Dashboard & Visualization)
* DAX (Measures & Calculations)
* Power Query (Data Cleaning)

---

## 🔧 Data Cleaning Steps

* Removed text units like "L" (Lakhs) from salary columns
* Converted salary values into numeric format
* Created calculated columns and measures
* Handled missing and inconsistent values

---

## 📈 Key Insights

* Certain states show consistently higher unemployment rates
* Data science roles offer competitive salaries across industries
* Growing demand for tech-related jobs indicates skill-based employment opportunities

---

## 🚀 How to Use

1. Download the `.pbix` file from this repository
2. Open using Power BI Desktop
3. Interact with filters, slicers, and visuals
